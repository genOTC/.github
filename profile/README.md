# genOTC

OTC derivatives pricing platform.

## Products

- **Volatility surface calibration** — local volatility model fitting from listed options market data.
- **Vanilla options pricing** — PDE-based pricer for European and American options.
- **Structured products pricing** — Monte Carlo engine for path-dependent and exotic payoffs.
- **Web application** — REST API and UI for running pricings, managing assets, and reviewing calibrations.
- **SDKs** — public client library wrapping the REST API for programmatic access.
- **Market data connectors** — ingestion pipelines for external providers.
